# TreeniLingo

TreeniLingo on Duolingo-tyylinen urheilusovellus punnerruksiin ja leuanvetoihin.

## Mitä sovellus tekee
- näyttää **haastepolun** (5 vaihetta)
- antaa **XP:tä**, nostaa **tasoa** ja seuraa **putkea**
- käyttää **sydämiä**: virheestä tai ohituksesta vähenee 1 sydän
- tarkistaa, että toistomäärä on vähintään haasteen vaatimus
- tallentaa tilanteen selaimen localStorageen

## Puhelinversio
- käyttöliittymä on tehty mobile-first-ajattelulla
- napit ovat isoja ja helposti painettavia
- layout skaalautuu puhelimesta työpöydälle

## Käyttö
1. Avaa `index.html` selaimessa.
2. Syötä toistot ja kirjaa oikea harjoite.
3. Suorita kaikki haasteet tai aloita alusta napista.
